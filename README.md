# IoT Greenhouse Monitoring System

## Overview

This project is an IoT-based smart greenhouse monitoring system developed using Arduino and sensors. It monitors temperature, humidity, and soil moisture levels in real time.

## Features

* Temperature Monitoring
* Humidity Monitoring
* Soil Moisture Monitoring
* Real-time Data Collection
* Smart Agriculture Support

## Hardware Components

* Arduino Board
* DHT11/DHT22 Sensor
* Soil Moisture Sensor
* Jumper Wires
* Breadboard

## Software Used

* Arduino IDE
* GitHub

## Project Objective

To improve greenhouse management by continuously monitoring environmental conditions and helping maintain optimal plant growth.
